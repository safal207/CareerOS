# Monetization Path

CareerOS monetization starts with a low-friction funnel:

```txt
Free Fit Audit -> Waitlist -> $19 Job Search Pack -> Pro Subscription / Done-with-you Offer
```

## Stage 1: Free Fit Audit

The free audit creates the first value moment.

User input:

- resume;
- vacancy.

User receives:

- match score;
- risk score;
- growth score;
- strong matches;
- missing skills;
- application strategy.

Goal:

> Prove that CareerOS can create clarity before asking for payment.

## Stage 2: Waitlist

The waitlist captures intent after the user sees value.

Current MVP behavior:

- frontend-only capture state;
- no backend persistence yet;
- validates demand and copy before implementation.

Future behavior:

- persist email;
- tag source and offer interest;
- send PLF-style launch sequence.

## Stage 3: $19 Job Search Pack

Tripwire offer:

> Apply to 10 better-fit jobs this week.

Included:

- 10 vacancy fit audits;
- 10 tailored application angles;
- resume headline rewrite;
- interview question pack;
- salary negotiation opener.

Why $19:

- low enough for impulse purchase;
- high enough to validate buyer intent;
- clear bridge from free audit to paid action.

## Stage 4: Subscription

Potential Pro plan:

- unlimited audits;
- saved vacancies;
- pipeline tracking;
- resume versions;
- interview prep;
- weekly job-search report.

Suggested price:

```txt
$29-$49/month
```

## Stage 5: Done-with-you offer

Premium service package:

> Package me for 10 target vacancies.

Suggested price:

```txt
$99-$299
```

Deliverables:

- rewritten positioning;
- 10 application drafts;
- interview prep;
- salary negotiation script.

## Key metrics

- audit completion rate;
- waitlist conversion rate;
- tripwire click rate;
- tripwire purchase rate;
- repeated audit usage;
- interview conversion reported by users.
