# Sample Vacancy

## Senior QA Engineer — FinTech Platform

We are looking for a Senior QA Engineer to join a fintech product team working on investment, reporting, and client-facing web systems.

## Responsibilities

- Test web and backend functionality for financial products.
- Validate REST API behavior and data consistency.
- Prepare and execute smoke, regression, integration, positive, and negative test cases.
- Investigate defects using logs, browser DevTools, and backend data.
- Collaborate with developers, analysts, product managers, and support teams.
- Improve QA processes and test documentation.

## Requirements

- 5+ years of QA experience.
- Strong manual testing background.
- Experience with API testing tools such as Postman or Swagger.
- SQL skills for data validation.
- Experience with fintech, banking, brokerage, or reporting products.
- Understanding of client-server architecture and microservices.
- Ability to write clear bug reports and test cases.

## Nice to have

- Kafka or event-driven systems experience.
- Docker basics.
- CI/CD familiarity.
- Python for automation.

## Conditions

- Remote or hybrid.
- Salary range not specified.
- Interview includes technical discussion and practical testing task.
